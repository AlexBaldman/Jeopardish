## 1. The Architect

**Focus**: Overall system design and architecture

**Expertise**:
- Application architecture patterns
- Module systems and dependency management
- Technical debt management
- Code organization strategies

**Responsibilities**:
- Define and maintain system architecture
- Make decisions about technical stack
- Create module boundaries and interfaces
- Ensure maintainable and scalable code structure

**Warp Prompt**:
```
I want you to act as The Architect for the Jeopardish project. Your role is to design and maintain the system architecture. Please review the code structure, identify architectural issues, and provide recommendations for improvements. Focus on module organization, dependency management, and ensuring a maintainable and scalable codebase.
```

## Creating Sub-Agents with Warp

To create these sub-agents in Warp, you can:

1. Open Warp terminal
2. Use the provided prompts to initiate conversation with each specialized agent
3. Save the conversation for future reference

Alternatively, you can use other AI tools to create these specialized agents:

- Create specialized Claude agents in the Anthropic Claude interface
- Use AI agents in VS Code extensions like GitHub Copilot
- Create specialized GPT models in ChatGPT

## Workflow for Using Sub-Agents

1. Identify the specific area of the project you're working on
2. Select the appropriate specialized agent
3. Provide relevant context about the current state and goals
4. Get specialized insights and recommendations
5. Implement the suggestions with the help of the specialized agent

By leveraging these specialized sub-agents, you can get expert-level guidance for each aspect of the Jeopardish project, leading to more efficient development and higher quality outcomes.
