🤖 Gemini AI Setup Guide for Jeopardish

Follow these steps to enable AI-powered features in Jeopardish, including AI-generated questions and witty Alex Trebek responses!

Step 1: Get a Free Gemini API Key

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Choose "Create API key in new project"
  5. Copy your API key (it looks like: AIzaSy...)

Note: The free tier includes 60 requests per minute, which is plenty for playing Jeopardish!

Step 2: Add Your API Key to Jeopardish

  1. Go back to Jeopardish
  2. Click the hamburger menu (☰) in the top right
  3. Click "Settings"
  4. Scroll down to "AI Host Settings"
  5. Paste your API key in the input field
  6. Click "Save"
  7. Make sure "Enable AI Host" is checked

Step 3: Test It Out!

Once your API key is saved:

  • You should see an AI-generated greeting from Alex Trebek
  • Questions will be rewritten with Trebek's personality
  • Correct/incorrect answers will get personalized responses
  • The ticker will show AI-generated comedy messages

⚠️ Important Security Note

Your API key is stored locally in your browser and is never sent to any server except Google's API. However, for production use, you should:

  • Use environment variables to store the API key
  • Set up API key restrictions in Google Cloud Console
  • Consider using a backend proxy to hide the key from client-side code

✅ Troubleshooting

If the AI features aren't working:

  • Check the browser console for error messages
  • Verify your API key is correct (no extra spaces)
  • Make sure you have internet connection
  • Try refreshing the page after saving your API key
  • Run testGemini() in the browser console to test the integration
